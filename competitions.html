Toronto Apartment-Market Analysis

GitHub Link

Description:
I initially conduct an exploratory analysis on the dataset to examine the distribution of features.
Then I cleaned the data by removing unnecessary columns and fill in null-cells by KNN (tractable due to size of dataset).
Then I extract features from domain-knowledge and common intuition.
Then I remove outliers based on exploratory analysis, and remove noise with an auto-encoder. Then add more features with polynomial augmentation.
Processed dataset is then normalized and split to train-test.
We use an X-model, and fine-tune the parameters with K-means elbow rule.
Conduct cross-validation by k-fold split.
Finally, I created the visualizations for this dataset to tell a story.
